

Project details

This is a web application that provides help to students looking for a suitable tutor for a particular course. The aim is to provide students with a platform where they can practice peer to peer tutoring, thereby giving a new facet to learning. The system suggests a list of tutors to a student on the basis of the tutor's existing profile and the system also helps in maintaining a schedule so as to avoid common hindrances due to same room being allocated or conflicting timings. Thereby providing the students with a hassle-free environment.

1.) Register
2.) Add classes to Profile
3.) Tutor or Be Tutored
